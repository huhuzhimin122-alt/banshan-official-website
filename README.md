# 半山官方网站

沉浸式官网首屏项目。

## 目标

从零搭建伴山官网，重点实现 Travel Next Level 类型的沉浸式视觉：3D 石门、张家界云海后景、滚动推进、平移后俯瞰。

## 开发

```bash
npm install
npm run dev
```

## 素材位置

- `public/models/stonegate.glb`：你们自己的石门模型
- `public/media/hidden-world.mp4`：即梦生成的无水印张家界云海视频
- `public/media/hidden-world-poster.jpg`：视频封面图

没有素材时页面会自动使用代码生成的临时山体背景。

## 预览

GitHub Pages 启用后会自动部署预览站点。
